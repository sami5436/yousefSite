Technology I Used

HTML is a markup language. It uses tags to define structure. So what are tags?

  • I used a tag to create this list. Tags are <p></p> and <ul></ul>.
  • Each tag has an opening and closing bracket.

These are the tags I have used so far:

  • p (paragraph)
  • hX (header, X being size)
  • ul (unordered list)
  • li (list item, always found in ul)
  • b (bold)
  • a(anchor element - hyperlink, creates a link using href to id in header) - where ever you want to have the hyperlink added, use the a and tie it with href.

Also, make sure you include the DOCTYPE Header at the top of your .html file. Make sure to include the html and body tags too (These 3 are required).
