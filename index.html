Technology I Used

HTML is a markup language. It uses tags to define structure. So what are tags?

  • I used a tag to create this list. Tags are <p></p> and <ul></ul>.
  • Each tag has an opening and closing bracket.
  • The tags I am using are p (paragraph), hX (header, X being size), b (bold), ul (unordered list), li (list item, always found in ul).

Also, make sure you include the DOCTYPE Header at the top of your .html file.
